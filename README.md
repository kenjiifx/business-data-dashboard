# Business Data Analysis Dashboard

Business Data Analysis Dashboard built with Python and Excel (Mar 2026).

## Project Overview

This project demonstrates an end-to-end business analysis workflow:

- Clean and structure raw business data using Python (`pandas`)
- Analyze trends, patterns, and performance metrics across key variables
- Build an interactive-style Excel dashboard with summaries, pivot-style tables, and charts
- Generate actionable insights to support business decision-making

## Tech Stack

- Python
- pandas
- Excel (`.xlsx` output via `xlsxwriter`)

## Project Structure

- `dashboard_builder.py` - Main analysis and dashboard generation script
- `data/business_data.csv` - Input dataset
- `output/business_data_dashboard.xlsx` - Generated Excel dashboard
- `output/actionable_insights.txt` - Generated insights summary
- `requirements.txt` - Python dependencies

## How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the dashboard builder:

```bash
python dashboard_builder.py
```

3. Check outputs in the `output/` folder.
